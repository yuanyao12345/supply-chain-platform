## 📋 第一部分：将代码放到GitHub（只需做一次）

### 步骤1：注册GitHub账号

1. 打开浏览器，访问：https://github.com/
2. 点击右上角的 **"Sign up"** 按钮
3. 填写注册信息：
   - Email（邮箱）：您的邮箱地址
   - Password（密码）：设置密码
   - Username（用户名）：例如 yuanyao12345
4. 点击 **"Create account"** 完成注册
5. 登录邮箱，点击验证链接激活账号

### 步骤2：创建GitHub仓库

1. 登录GitHub
2. 点击右上角的 **"+"** 按钮，选择 **"New repository"**
3. 填写仓库信息：
   - Repository name（仓库名）：`supply-chain-platform`
   - Description（描述）：`花湖国际机场供应链金融平台`
   - 选择 **"Public"**（公开）
   - 勾选 **"Add a README file"**
4. 点击 **"Create repository"** 创建仓库

### 步骤3：安装Git工具

**Windows用户：**
1. 访问：https://git-scm.com/download/win
2. 下载并安装Git
3. 安装过程中一直点击 **"Next"** 即可

**Mac用户：**
1. 打开终端
2. 输入：`git --version`
3. 如果没有安装，会提示安装，按提示操作

### 步骤4：配置Git

1. 打开命令提示符（Windows）或终端（Mac）
2. 配置用户名和邮箱：
```bash
git config --global user.name "您的用户名"
git config --global user.email "您的邮箱"
```

### 步骤5：将本地代码推送到GitHub

1. **打开项目文件夹**
   - 打开文件管理器
   - 进入项目目录：`c:\Users\yuany\Documents\trae_projects\Supply-Chain-Plat`

2. **右键点击空白处，选择 "Git Bash Here"**

3. **初始化Git仓库**（如果还没初始化）：
```bash
git init
```

4. **添加所有文件到Git**：
```bash
git add .
```

5. **提交文件**：
```bash
git commit -m "初始提交"
```

6. **连接GitHub仓库**：
```bash
git remote add origin https://github.com/您的用户名/supply-chain-platform.git
```

7. **推送到GitHub**：
```bash
git push -u origin main
```

8. **输入GitHub用户名和密码**
   - 用户名：您的GitHub用户名
   - 密码：您的GitHub密码或Token

9. **等待推送完成**
   - 看到 "Writing objects: 100%" 说明成功

10. **验证**
    - 打开浏览器，访问：`https://github.com/您的用户名/supply-chain-platform`
    - 看到文件列表说明成功！

## 📋 第二部分：首次部署到PythonAnywhere（只需做一次）

### 步骤1：注册PythonAnywhere账号

1. 打开浏览器，访问：https://www.pythonanywhere.com/
2. 点击右上角的 **"Sign up"** 按钮
3. 填写注册信息：
   - Username（用户名）：建议使用 yuanyao
   - Email（邮箱）：您的邮箱地址
   - Password（密码）：设置密码
4. 点击 **"Create account"** 完成注册
5. 登录邮箱，点击验证链接激活账号

### 步骤2：创建Web应用

1. 登录PythonAnywhere
2. 点击顶部菜单的 **"Web"**
3. 点击 **"Add a new web app"**
4. 选择 **"Manual configuration"**
5. 选择 **"Python 3.9"**
6. 点击 **"Next"** 完成创建

### 步骤3：运行自动部署脚本

1. 点击顶部菜单的 **"Consoles"**
2. 点击 **"Bash"** 打开Bash控制台
3. 复制以下命令，粘贴到控制台，按回车执行：

```bash
cd ~
rm -rf supplychain
git clone https://github.com/yuanyao12345/supply-chain-platform.git supplychain
cd supplychain
pip install -r requirements.txt
python -c 'from app import app, db; app.app_context().push(); db.create_all(); print("Database created!")'
python -c 'from app import app, db, Admin; app.app_context().push(); admin = Admin.query.filter_by(username="admin").first(); if not admin: admin = Admin(username="admin", password="admin123"); db.session.add(admin); db.session.commit(); print("Admin account created!"); else: print("Admin account already exists, skipping...")'
echo "代码部署完成！"
```

4. 等待命令执行完成（看到"代码部署完成！"提示）

### 步骤4：配置Web应用

1. 点击顶部菜单的 **"Web"**
2. 找到您的应用：`yuanyao.pythonanywhere.com`
3. 找到 **"Code"** 部分：
   - **Source code**：改为 `/home/yuanyao/supplychain`
   - **Working directory**：改为 `/home/yuanyao/supplychain`
4. 找到 **"WSGI configuration file"** 链接，点击进入
5. 删除所有内容，粘贴以下代码：

```python
import sys
import os

# 添加项目路径
path = '/home/yuanyao/supplychain'
if path not in sys.path:
    sys.path.append(path)

# 设置环境变量
os.environ.setdefault('FLASK_ENV', 'production')
os.environ.setdefault('SECRET_KEY', 'your-secret-key-change-this')

# 导入应用
from app import app as application
```

6. 点击 **"Save"** 保存
7. 点击 **"Reload"** 按钮重启应用
8. 等待几秒钟，点击链接访问您的网站

### 步骤5：验证部署

1. 访问：https://yuanyao.pythonanywhere.com
2. 看到首页说明部署成功！
3. 测试以下功能：
   - 企业注册
   - 银行注册
   - 融资申请
   - 后台管理（用户名：admin，密码：admin123）
   - 融资审批

## 🔄 第三部分：修改网站（可多次操作）

### 完整流程图

```
本地修改代码 → 保存文件 → git add → git commit → git push → 
PythonAnywhere: git pull → Reload → 查看效果
```

### 场景1：修改文字内容

**示例：修改首页标题**

1. **在本地修改**
   - 打开文件：`templates/index.html`
   - 找到 `<title>` 标签
   - 修改文字内容
   - 保存文件（Ctrl+S）

2. **提交到GitHub**
   ```bash
   git add templates/index.html
   git commit -m "修改首页标题"
   git push origin main
   ```

3. **在PythonAnywhere更新**
   ```bash
   cd ~/supplychain
   git pull origin main
   ```
   然后点击 **"Reload"** 按钮

4. **查看效果**
   - 访问网站，查看修改是否生效
   - 如果没有生效，按Ctrl+F5强制刷新

### 场景2：修改颜色

**示例：修改导航栏颜色**

1. **在本地修改**
   - 打开文件：`templates/index.html`
   - 找到 `.header` 样式
   - 修改颜色代码
   - 保存文件（Ctrl+S）

2. **提交到GitHub**
   ```bash
   git add templates/index.html
   git commit -m "修改导航栏颜色"
   git push origin main
   ```

3. **在PythonAnywhere更新**
   ```bash
   cd ~/supplychain
   git pull origin main
   ```
   然后点击 **"Reload"** 按钮

4. **查看效果**
   - 访问网站，查看颜色是否改变

### 场景3：添加新页面

**示例：添加"帮助中心"页面**

1. **创建页面文件**
   - 复制 `templates/about.html`
   - 重命名为 `templates/help.html`
   - 修改页面内容
   - 保存文件

2. **添加路由**
   - 打开 `app.py`
   - 在 `# 关于我们` 下方添加：
   ```python
   # 帮助中心
   @app.route('/help')
   def help():
       return render_template('help.html')
   ```
   - 保存文件

3. **添加导航链接**
   - 打开 `templates/index.html`
   - 在导航栏添加：
   ```html
   <a href="{{ url_for('help') }}">帮助中心</a>
   ```
   - 保存文件

4. **提交到GitHub**
   ```bash
   git add templates/help.html templates/index.html app.py
   git commit -m "添加帮助中心页面"
   git push origin main
   ```

5. **在PythonAnywhere更新**
   ```bash
   cd ~/supplychain
   git pull origin main
   ```
   然后点击 **"Reload"** 按钮

6. **查看效果**
   - 访问网站，点击"帮助中心"链接
   - 查看新页面是否正常显示

## ❌ 常见错误及解决方法

### 错误1：git push 失败
**症状**：提示 "failed to push"
**解决**：
```bash
git pull origin main
git push origin main
```

### 错误2：PythonAnywhere上 git pull 失败
**症状**：提示 "error: Your local changes..."
**解决**：
```bash
cd ~/supplychain
git stash
git pull origin main
git stash pop
```

### 错误3：网站显示500错误
**症状**：页面显示 "Internal Server Error"
**解决**：
1. 检查错误日志：在PythonAnywhere的Web页面点击 "Error log"
2. 修复问题后点击 "Reload"

### 错误4：修改不生效
**症状**：修改后网站没有变化
**解决**：
1. 确认已执行 `git push origin main`
2. 在PythonAnywhere执行 `git pull origin main`
3. 点击 "Reload" 按钮
4. 清除浏览器缓存（Ctrl+F5）

### 错误5：git commit 提示 "Please tell me who you are"
**症状**：提示需要配置用户名和邮箱
**解决**：
```bash
git config --global user.name "您的用户名"
git config --global user.email "您的邮箱"
```

### 错误6：git push 提示 "Permission denied"
**症状**：没有权限推送
**解决**：
1. 检查GitHub仓库是否为自己的
2. 检查用户名和密码是否正确
3. 使用Token代替密码（GitHub设置 → Developer settings → Personal access tokens）
